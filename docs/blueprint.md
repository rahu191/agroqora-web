# **App Name**: AgriGenius AI

## Core Features:

- Pest & Disease Diagnosis: Upload leaf images to diagnose plant health using Vision Transformer (ViT) or EfficientNet AI models, recommending appropriate treatments.
- Fertilizer Recommendation: Analyze soil nutrient data via IoT sensors and recommend optimal fertilizer applications using XGBoost or TabNet AI models.
- Crop Planning Optimization: Suggest sowing and harvest schedules based on historical yields, weather data, and market trends, utilizing Reinforcement Learning and Retrieval-Augmented Generation (RAG) with Vertex AI Search. The system will act as a planning tool by helping the user decide the most appropriate time to act.
- Market Trend Analysis: Analyze market and seasonal trends to recommend the most profitable crops using Prophet and Graph Neural Networks (GNNs).
- Automated IoT Sprayer Activation: Automatically activate IoT sprayers based on AI analysis and predefined rules, optimizing fertilizer and pesticide application.
- User-Friendly Mobile App: Provide a Flutter-based mobile app interface for farmers with features like image upload, text/chat interaction, voice prompts, and calendar-based farm planning.
- Real-time Monitoring and Alerts: Monitor sensor data and model performance, providing real-time alerts and notifications via Firebase or SMS for critical events like pest detection or soil imbalances.

## Style Guidelines:

- Primary color: Earthy green (#6B8E23) to evoke nature and growth.
- Background color: Light beige (#F5F5DC) for a clean and natural feel.
- Accent color: Warm yellow (#FFC107) to highlight important information and calls to action.
- Body and headline font: 'PT Sans' for a modern and readable design.
- Use clear, descriptive icons for different features (e.g., a leaf for pest diagnosis, a droplet for irrigation).
- Implement a clean, intuitive layout optimized for mobile devices, ensuring easy navigation and accessibility.
- Use subtle animations to provide feedback on user actions and guide the user through the app.
